  class_id: uuid (FK -> classes.id, nullable)

Tabelle progress_entries:
  id: uuid (PK)

  operation_type: text ('addition'|'subtraktion'|'multiplikation'|'division'|'minigame_redeem')
  grade: int (1-4)
  correct: boolean
  points_earned: int (default 0)
  created_at: timestamptz

Beziehungen:
  schools 1--* classes (school_id)
  classes *--1 auth.users (teacher_id)
  profiles *--1 classes (class_id)
  profiles 1--1 auth.users (user_id)
  progress_entries *--1 auth.users (child_id)

5. **BPMN-Prozessdiagramm: Uebungsprozess** (per D-12):

   Da Mermaid kein natives BPMN unterstuetzt, verwende ein Mermaid flowchart (flowchart TD oder LR) das BPMN-Elemente simuliert:
   - Start-Event (Kreis): Kind oeffnet Ueben-Seite
   - Task (Rechteck): Aufgabe wird generiert
   - Task: Kind gibt Antwort ein
   - Gateway (Raute): Antwort korrekt?
     - Ja: Punkte vergeben -> Schwierigkeit pruefen -> naechste Aufgabe
     - Nein: Korrekte Antwort zeigen -> naechste Aufgabe
   - Gateway (Raute): Punkte-Schwelle erreicht?
     - Ja: Mini-Game freischalten
     - Nein: Weiter ueben
   - End-Event: Session beendet / Mini-Game gespielt

   Diagramm-Syntax:
   ```mermaid
   flowchart TD
     Start((Start)) --> Generate[Aufgabe generieren]
     Generate --> Display[Aufgabe anzeigen]
     Display --> Input[Antwort eingeben]
     Input --> Check{Korrekt?}
     Check -->|Ja| Points[Punkte vergeben]
     Check -->|Nein| ShowCorrect[Korrekte Antwort zeigen]
     Points --> Difficulty[Schwierigkeit anpassen]
     ShowCorrect --> NextQ[Naechste Aufgabe]
     Difficulty --> Threshold{Punkte-Schwelle erreicht?}
     Threshold -->|Nein| NextQ
     NextQ --> Generate
     Threshold -->|Ja| Unlock[Mini-Game freischalten]
     Unlock --> Game[Mini-Game spielen]
     Game --> End((Ende))
   ```

   Unter dem Diagramm: Erklaerung (3-5 Saetze) die den Gamification-Loop beschreibt:
   - Kern-Loop: Aufgabe -> Antwort -> Feedback -> naechste Aufgabe
   - Motivations-Mechanik: Punkte als Fortschrittsindikator
   - Belohnungs-Gate: Mini-Game nur bei genuegend Punkten
   - Schwierigkeits-Adaption: automatische Anpassung nach Leistung

Sprache: Formelles Deutsch.
