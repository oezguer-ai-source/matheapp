**Datei 1: lib/utils/relative-date.ts** (per D-10)

Export: `formatRelativeDate(isoDate: string | null): string`
- null -> "Keine Aktivitaet"
- Heute -> "Heute"
- Gestern -> "Gestern"
- 2-6 Tage -> "vor X Tagen"
- 7-30 Tage -> "vor X Tagen" (gleiche Logik)
- Aelter als 30 Tage -> deutsches Datum "TT.MM.JJJJ"
- Vergleich basiert auf Kalender-Tagen (nicht 24h-Perioden): Start of Day in lokaler Zeitzone

Export: `isInactive(isoDate: string | null): boolean` (per D-09)
- true wenn isoDate null oder aelter als 7 Tage
- Wird fuer die visuelle Hervorhebung in der Tabelle verwendet
