Erstelle `components/child/balloon-game.tsx` — die Haupt-Spielkomponente per D-01 bis D-09:

"use client" Direktive

Imports:
- useState, useEffect, useCallback, useRef aus react
- Balloon aus ./balloon
- GameOverScreen aus ./game-over-screen
- startGameAction aus @/app/(child)/kind/spiel/actions (fuer Punkt-Abzug)

Interface BalloonGameProps: `{ currentPoints: number }` (wird von page.tsx uebergeben)

Balloon-Typ (intern):
```typescript
interface BalloonData {
  id: number;
  x: number;       // horizontal position 5-90 (%)
  color: string;    // Tailwind color class
  speed: number;    // animation duration 4-8 seconds
  popped: boolean;
}
```

Konstanten:
- GAME_DURATION = 75 (Sekunden, Mitte von 60-90 per D-03)
- SPAWN_INTERVAL = 800 (ms zwischen Ballon-Spawns)
- MAX_BALLOONS = 12 (gleichzeitig, Pitfall 3)
- COLORS Array: `["bg-child-yellow", "bg-child-green", "bg-child-blue", "bg-child-red", "bg-pink-400", "bg-purple-400", "bg-orange-400"]` (D-07, bunte Farben)

Game States: `"idle" | "starting" | "playing" | "over"`

State:
- gameState: "idle" | "starting" | "playing" | "over" (startet mit "idle")
- balloons: BalloonData[] (startet leer)
- score: number (startet 0)
- timeLeft: number (startet GAME_DURATION)
- error: string | null

Refs:
- nextBalloonId: useRef(0) (fuer eindeutige IDs)
- spawnIntervalRef: useRef fuer Cleanup
- timerIntervalRef: useRef fuer Cleanup

Funktionen:

1. `handleStartGame`:
   - Setze gameState auf "starting"
   - Rufe `startGameAction()` auf (Server Action fuer Punkt-Abzug per D-12)
   - Bei Fehler: setze error-State und gameState zurueck auf "idle"
   - Bei Erfolg: setze gameState auf "playing", reset score/timeLeft/balloons

2. `spawnBalloon` (useCallback):
   - Nur wenn balloons.length < MAX_BALLOONS
   - Erstelle neues BalloonData mit:
     - id: nextBalloonId.current++ 
     - x: Math.random() * 80 + 5 (5% bis 85%)
     - color: zufaellig aus COLORS
     - speed: Math.random() * 4 + 4 (4-8 Sekunden)
     - popped: false
   - Fuege zu balloons State hinzu

3. `handlePop(id: number)` (useCallback):
   - Finde Ballon mit id in balloons
   - Wenn bereits popped: return (kein Doppel-Klick)
   - Setze popped: true fuer den Ballon
   - Erhoehe score um 1 (D-06)
   - Nach 300ms (Pop-Animation-Dauer): entferne Ballon aus State

4. useEffect fuer Timer (wenn gameState === "playing"):
   - setInterval alle 1000ms: timeLeft - 1
   - Wenn timeLeft <= 0: setze gameState auf "over", clear alle Intervals
   - Cleanup: clearInterval

5. useEffect fuer Ballon-Spawning (wenn gameState === "playing"):
   - setInterval alle SPAWN_INTERVAL ms: spawnBalloon aufrufen
   - Cleanup: clearInterval

6. useEffect fuer Ballon-Cleanup (entferne Ballons die oben angekommen sind):
   - Alle 1000ms: entferne Ballons deren Lebenszeit > speed (basierend auf creation time oder einfach: entferne nicht-geplatzte Ballons nach ihrer speed-Dauer)
   - Alternative: Nutze onAnimationEnd Event auf dem Balloon-Element statt Timer-basiertem Cleanup

Rendering:

- gameState === "idle" oder "starting":
  - Zentrierter Container mit "Ballonplatzen!"-Heading (text-5xl)
  - Kurze Beschreibung: "Platze so viele Ballons wie moeglich!" (text-2xl)
  - "Spiel starten"-Button (bg-child-green, text-white, text-3xl, h-16, rounded-2xl)
  - Bei "starting": Button zeigt "Starte..." und ist disabled
  - Bei error: Fehlermeldung in rot

- gameState === "playing":
  - Fester Container: `relative w-full h-[80vh] overflow-hidden bg-gradient-to-b from-sky-200 to-sky-400 rounded-2xl`
  - Score-Counter oben links: `absolute top-4 left-4 text-3xl font-bold bg-white/80 rounded-xl px-4 py-2` — zeigt Score (D-08)
  - Timer oben rechts: `absolute top-4 right-4 text-3xl font-bold bg-white/80 rounded-xl px-4 py-2` — zeigt timeLeft in Sekunden
  - Map ueber balloons-Array: rendere `<Balloon>` fuer jeden nicht-entfernten Ballon

- gameState === "over":
  - Rendere `<GameOverScreen score={score} />` (D-09)

WICHTIG:

- KEINE Server-Kommunikation waehrend des Spielens — nur VOR dem Spiel (startGameAction) per D-04
- Der Ballon-Score ist rein visuell, NICHT persistent
